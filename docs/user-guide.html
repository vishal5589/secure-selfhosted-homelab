How to Request
Movies & TV Shows

Everything you need to know about requesting content on our Discord — it's quick, easy, and totally free of limits!

🚀
How It Works
1

Head to the Channel

Go to the requests channel in our Discord server.

2

Type a Command

Use a slash command like /request english-movie to get started.

3

Search & Confirm

The bot will ask for the title — pick the right one and confirm!

4

Admin Approval

Your request is reviewed by an admin in Overseerr. It needs to be approved before downloading begins.

5

Sit Back & Enjoy

Once approved, it downloads automatically. You'll be notified when it's ready on Plex. 🍿

Tips & Good to Know
♾️

No Request Limits

Request as much as you like — there's no daily or weekly cap. Go wild!

📅

Include the Year

If a title has remakes (e.g. The Lion King), adding the release year helps find the exact one.

🌍

Foreign vs Tamil

Use /request foreign-movie for non-English, non-Tamil films like Korean, Spanish, or French movies.

📺

All TV Uses /tv-show

One command covers everything — dramas, anime, reality TV, miniseries. The bot will ask you about seasons.

Bot Offline?

Use /ping to check. If no reply, give it a few minutes or ping an admin.

⏱️

Be Patient

Downloads can take anywhere from a few minutes to a few hours depending on availability and size.

📌
Quick Reminder

All commands are used directly in the Discord request channel. No registration, no sign-in, no DM needed — just type a command and go!
